Con grid aninados (ponerle grid a la subgrid) se crean grid dentro de grid pero en realidad estas son independientes

Si a un elemento de la sub grid le ponemos un tamaño muy alto, este sobrepasará incluso las medidas de su padre grid

Con subgrid los hijos si perteneceran a la grid padre y conservaran las medidas de este o se ajustaran dependiendo del jefesito
